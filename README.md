# NEURONEX - Futuristic Global News Portal

**NEURONEX** is a next-generation, AI-inspired news portal delivering real-time global intelligence in a sleek, high-tech interface. Designed for a futuristic, immersive experience, NEURONEX aggregates news from multiple APIs, including NewsAPI, GNews, and World News API, providing dynamic breaking news, category-based feeds, and AI-style article suggestions based on user behavior.

---

## Features

- **Dynamic Breaking News:** Headlines update randomly every visit.  
- **Multi-category System:** Global, Economy, Sports, Entertainment, Politics.  
- **AI-Style Suggestions:** Personalized recommended articles based on user reading patterns.  
- **Search Functionality:** Query news across multiple APIs in real-time.  
- **Dark/Light Theme:** User-selectable with smooth transition.  
- **Fully Responsive:** Works seamlessly on mobile, tablet, and desktop.  
- **Futuristic UI:** Animated cards, neon accents, glassmorphism design, and cyber-inspired visual elements.  
- **Randomized Feed:** Every refresh presents new, shuffled articles for a dynamic experience.  

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Responsive Grid & Animations  
- **APIs Integrated:**  
  - NewsAPI.org  
  - GNews API  
  - World News API  
- **Deployment Ready:** Can be hosted on Vercel as a static site.
